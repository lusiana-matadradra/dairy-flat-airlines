# Dairy Flat Airlines

A full-stack airline booking system built with Next.js, MongoDB, TypeScript, and Tailwind CSS.

## Features

- View flight schedules
- Search flights
- Create passenger bookings
- Passenger booking lookup
- Cancel bookings
- Live seat availability updates
- MongoDB database integration
- Multiple weeks of generated schedules

## Technologies Used

- Next.js 16
- TypeScript
- MongoDB Atlas
- Tailwind CSS
- React

## How To Run

Install dependencies:

npm install

Start development server:

npm run dev

Seed database:

npx tsx src/scripts/seed.ts
